# Spatiotemporal Water Quality Assessment (Karun, Dez, Karkheh)

Python scripts for the paper: **"Integrated Water Quality Assessment of Karun, Dez, and Karkheh Basins"**.

## How to run
- Install dependencies: `pip install -r requirements.txt`
- Run scripts in `src/` directory.
