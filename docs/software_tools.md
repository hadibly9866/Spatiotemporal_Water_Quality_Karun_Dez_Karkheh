# Software and Tools Used

- **Programming Language:** Python 3.9+
- **Key Libraries:** 
    - `Pandas` & `NumPy` (Data Processing)
    - `Pymannkendall` (Trend Analysis)
    - `PySAL` & `Libpysal` (Spatial Analysis)
    - `Matplotlib` & `Seaborn` (Visualization)
    - `Mgwr` (Multiscale Geographically Weighted Regression)
- **GIS Software:** ArcGIS Pro / QGIS (for final map rendering)
- **Hydrochemistry:** AquaChem (for Piper diagrams)
